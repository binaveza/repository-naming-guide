# repository-naming-guide

Приветствую, путник! В моих репозиториях можно запутаться, но если тебе интересно разобраться, можешь использовать этот шаблон:

| Префикс | Суффикс | Описание | Пример |
|---------|---------|----------|---------|
| `iqs-` | `{курс}` | Репозитории пройденных курсов с решенной практикой | [`iqs-statistics-in-python`](https://github.com/binaveza/iqs-statistics-in-python) |
| `tt-` | `{должность}` | Решения тестовых заданий при трудоустройстве | [`tt-product-analyst`](https://github.com/binaveza/tt-product-analyst) |
| `{технология}` | `-little-projects` | Сборник решений на определенных языках и технологиях | [`1c-little-projects`](https://github.com/binaveza/1c-little-projects) |

Актуальные репозитории я обязательно креплю в таблицу README на главной странице, следи за обновлениями :)

#### [Переход ко всем репозиториям](https://github.com/binaveza?tab=repositories)

Hello and welcome! With so many repositories, it can be easy to lose track. 
To help you navigate, here's a template you can use:

| Prefix | Suffix | Description | Example |
|--------|--------|-------------|---------|
| `iqs-` | `{course}` | Repositories of completed courses with solved practice tasks | [`iqs-statistics-in-python`](https://github.com/binaveza/iqs-statistics-in-python) |
| `tt-` | `{position}` | Solutions for test assignments during job applications | [`tt-product-analyst`](https://github.com/binaveza/tt-product-analyst) |
| `{technology}` | `-little-projects` | Collection of solutions in specific languages and technologies | [`1c-little-projects`](https://github.com/binaveza/1c-little-projects) |

Active repositories are pinned in the README table on the main page. 
Follow for updates! :)
